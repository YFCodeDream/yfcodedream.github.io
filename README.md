# Yifei Gao Personal Homepage

This is a static GitHub Pages homepage generated from the local CV, publication links, and introduction materials.

## Files

- `index.html`: homepage content
- `styles.css`: responsive page styling
- `script.js`: navigation, language switching, and publication filtering
- `assets/images/`: research visuals extracted from the introduction deck
- `assets/downloads/`: public downloadable CV

## Deploy To GitHub Pages

Create a repository named `yfcodedream.github.io`, then push the public site files to the repository root:

```bash
git init
git add .
git commit -m "Launch personal homepage"
git branch -M main
git remote add origin git@github.com:YFCodeDream/yfcodedream.github.io.git
git push -u origin main
```

GitHub Pages will serve the site at:

```text
https://yfcodedream.github.io
```
